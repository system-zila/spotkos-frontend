Prompt Proper untuk Figma AI – Website SpotKos

Prompt:

Create a modern, clean, and professional landing page design for a boarding house recommendation platform called SpotKos.

About the Platform:
SpotKos is a web platform that helps students, workers, and newcomers find comfortable, safe, and affordable boarding houses (kos) in major cities in Indonesia such as Jakarta, Bandung, Yogyakarta, and Surabaya.

The platform focuses on trusted recommendations, verified listings, transparent pricing, and strategic locations near campuses, offices, and public transportation.

The design should feel modern, trustworthy, minimalist, and startup-style, similar to travel or property platforms.

Use a clean grid layout, rounded cards, soft shadows, and clear visual hierarchy to ensure the interface is easy to navigate.

Design Style

Modern SaaS / startup-style interface

Minimalist and clean UI

Card-based design for listings

Large hero search bar for property search

Friendly but professional appearance

Color Palette:

Primary: Soft Blue

Secondary: White

Accent: Light Gray

Typography:

Modern sans-serif font

Clear hierarchy for headings, subheadings, and body text
